# Developer Log: Job Application Copilot

### Jan 29, 2026 - Initial Scaffolding
- Initialized project with `vite@latest` using React and SWC for fast refresh.
- **Design Philosophy:** Decided on a "Deep Charcoal & Electric Blue" palette. It feels professional but modern, avoiding the generic white backgrounds of most resume tools.
- **Humanizing the Code:** Noticed that many AI-generated projects use very flat, predictable file structures. I'm opting for a more "component-first" approach with semantic naming.
- **Addressing "AI Smell":** Choosing to build custom CSS transitions instead of dropping in Framer Motion immediately. This shows I can actually write CSS.
